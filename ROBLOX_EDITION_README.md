# 🎮 FINUTURO 3D - ROBLOX Edition

## 📋 Descripción

**Finuturo 3D** es un simulador de educación financiera desarrollado en **ROBLOX** que enseña conceptos de finanzas a estudiantes de secundaria de forma divertida e interactiva.

El jugador comienza con poco dinero y debe aprender a administrar sus finanzas para convertirse en un empresario exitoso.

## 🚀 Instalación en ROBLOX

### Paso 1: Crear un nuevo lugar
- Abre ROBLOX Studio
- Click en "New"
- Selecciona "Baseplate"

### Paso 2: Importar Scripts
- Descarga los scripts de GitHub
- Cópialos en:
  - **ServerScriptService** → Scripts del servidor
  - **StarterGui** → Scripts de UI
  - **ReplicatedStorage** → Módulos compartidos

### Paso 3: Configurar DataStore
- Activa **API Access** en Game Settings
- Habilita **Read/Write Access**

### Paso 4: Publicar el Juego
- Click en "File" → "Save to Roblox"
- Dale un nombre y descripción
- ¡Listo para jugar!
